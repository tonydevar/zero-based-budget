# zero-based-budget
Antigravity project: Build a zero-based budgeting web app inspired by YNAB (You Need A Budget). The core philosophy is 'give every dollar a job' — all income must be assigned to budget categories until the 'Ready to Assign' amount reaches $0.Tech stack: React 18 with Vite for the frontend, Tailwind CSS for styling, Node.js with Express for the back
